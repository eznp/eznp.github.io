java-urldns链分析

URLDNS链分析

这个链子可以用来判断是否有反序列化漏洞，其本身没有什么攻击性，也比较简单，就是进行一次DNS请求。

原理

java.util.HashMap 重写了 readObject, 在反序列化时会调用 hash 函数计算 key 的 hashCode.而 java.net.URL 的 hashCode 在计算时会调用 getHostAddress 来解析域名, 从而发出 DNS 请求.
